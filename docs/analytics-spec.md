# Analytics Specification

Analytics consume canonical records only.

## Implemented Modules
- THI / heat stress: `packages/analytics/thi.py`
- Basic herd anomaly detection: `packages/analytics/anomaly.py`
- Market/economic trend summary: `packages/analytics/market.py`
- Data quality diagnostics: `packages/analytics/data_quality.py`
